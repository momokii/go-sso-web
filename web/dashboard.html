Echo Notes - App Information
Echo Notes – Capture & Summarize Your Meetings

An intelligent web app for recording, transcribing, and summarizing real-life conversations.

Project Overview

Echo-Notes is a personal project designed to streamline the process of capturing and summarizing real-world meetings. Built with Golang using the Fiber framework, PostgreSQL, and a sleek interface powered by HTML, Bootstrap, and jQuery, this app integrates cutting-edge AI technology to deliver efficient meeting insights.

Key Features
  • Seamless Audio Recording:
    Record real-life meetings directly within the web app with high-quality audio capture.
  • AI-Powered Transcription:
    Convert speech to text in real time using Whisper (OpenAI's advanced speech-to-text model).
  • Smart Summarization:
    Generate two types of summaries with GPT-based AI:
    • Simple Summary – A concise version highlighting key points.
    • Comprehensive Summary – A detailed breakdown for deeper insights.
  • Secure & Personalized:
    Save, organize, and revisit meeting records with user authentication and session management.
Why Echo-Notes?

Echo-Notes is more than just a recording tool, it's an intelligent assistant for busy professionals, teams, and students. By leveraging AI-powered transcription and summarization, it enhances productivity and ensures no important detail is missed.

Gabut Project – Unleashing the Power of Large Language Models (LLMs)

Exploring the possibilities of LLMs by blending creativity and technology into practical applications.

Project Overview

The Gabut Project is a personal initiative aimed at experimenting with the potential of Large Language Models (LLMs) across diverse domains. This project leverages technologies from providers like OpenAI and Anthropic to deliver innovative and engaging features.

Key Features
  • Medium Account Roast:
    A playful yet insightful analysis of your Medium account to improve your online presence with humorous feedback.
  • Baku Hantam X Website Roast/Analysis:
    Deep dive into Baku Hantam X, a website that compiles summaries of 'Baku Hantam' tweets from the X platform. It attempts to scrape, roast, and condense them into concise summaries for users.
  • Butterfly Effect Stories Generator:
    Generate imaginative "What if?" scenarios, creating branching stories that showcase the creativity and adaptability of LLMs.
  • Creative Content Generator:
    Simplify content creation by generating high-quality articles, captions, and other creative materials tailored to user needs.

The Gabut Project is a testament to curiosity-driven learning, pushing the boundaries of what's possible with LLMs. It demonstrates how cutting-edge technologies can be applied to real-world use cases. Go LLM Bridge
Go LLM Wrapper – A Pure Go Interface for LLM APIs

Streamline your integration with advanced LLM capabilities including text, vision, and audio.

Project Overview

The Go LLM Wrapper provides a streamlined, pure Go interface for interacting with leading LLM APIs such as Anthropic’s Claude and OpenAI. Designed with zero external dependencies, this lightweight wrapper supports an extensive range of functionalities—ranging from text processing and vision tasks to advanced audio processing. It simplifies the integration of cutting-edge language model capabilities into your Go applications.

Key Features
  • 🚀 Pure Go Implementation: Built with pure Go and zero external dependencies for optimal performance.
  • 💬 Multiple Provider Support: Seamlessly integrates with both Anthropic and OpenAI.
  • 👁️ Comprehensive Capabilities: Supports both text-based operations and vision functionalities.
  • 🎙️ Audio Processing: Offers state-of-the-art text-to-speech and speech-to-text features.
  • 🖼️ Image Generation: Leverages DALL-E for creative image generation.
  • ⚡ Minimal and Efficient: Optimized for speed and low resource consumption.
  • 🛠️ Flexible Configuration: Provides customizable options tailored to each provider.
Why This Project Matters?

This project exemplifies modern API design with a focus on efficiency and versatility. By delivering a pure Go solution for integrating advanced LLM functionalities, it empowers developers to build smarter applications, accelerate innovation, and reduce development overhead.

GinVExpress - App Information
Thesis Project – API Performance Comparison: Golang Gin vs. Node.js Express

A comprehensive performance analysis of API frameworks with SQL and NoSQL databases.

Project Overview

My final university project explores the performance differences between APIs developed with Golang Gin and Node.js Express, connected to two distinct database types—SQL (PostgreSQL) and NoSQL (MongoDB). The study involves four comprehensive test cases, with all APIs implemented as URL shortener services to ensure consistent functionality and fair benchmarking.

Key Focus Areas
  • Testing Environment:
    Both APIs are deployed on Google Cloud Run, ensuring identical environments to eliminate external factors influencing the results.
  • Performance Testing:
    Utilizing the K6 tool, tests focus on critical metrics such as iterations per second, CPU utilization, memory consumption, and response times under various workloads.
  • Key Insights:
    Unveiling the strengths and weaknesses of each framework in handling SQL and NoSQL databases, providing practical insights for developers when choosing between Golang and Node.js.
Why This Thesis Matters?

This project combines academic rigor with real-world applicability, showcasing technical expertise in backend development, performance testing, and cloud deployment. It highlights my ability to analyze complex systems and deliver actionable insights—an essential skill for creating efficient, scalable web applications.
