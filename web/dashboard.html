Thesis Project - App Information
GinVExpress

API Performance Comparison: Golang Gin vs. Node.js Express - A comprehensive analysis with SQL and NoSQL databases.

Technologies
  • Golang (Gin)
  • Node.js (Express)
  • PostgreSQL & MongoDB
  • Google Cloud Run
Testing Tools
K6 Performance Testing
CPU Profiling
Memory Analysis
Response Time Metrics
Key Focus Areas

Testing Environment

Both APIs are deployed on Google Cloud Run, ensuring identical environments to eliminate external factors influencing the results.

Performance Testing

Utilizing the K6 tool, tests focus on critical metrics such as iterations per second, CPU utilization, memory consumption, and response times.

URL Shortener Implementation

All APIs implement identical URL shortener services to ensure consistent functionality and fair benchmarking across frameworks.

Database Comparison

Side-by-side analysis of SQL (PostgreSQL) and NoSQL (MongoDB) performance with both Gin and Express frameworks.

Why This Thesis Matters

This project combines academic rigor with real-world applicability, showcasing technical expertise in backend development, performance testing, and cloud deployment. It highlights my ability to analyze complex systems and deliver actionable insightsan essential skill for creating efficient, scalable web applications. The findings provide valuable guidance for developers choosing between Golang and Node.js frameworks.
